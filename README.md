# Honey Do (Test)

Monorepo containing the Honey Do application:

- **Backend** — Spring Boot (Gradle, Java 21) at the repository root (`build.gradle`, `src/`)
- **Frontend** — Vue 3 + TypeScript + Vite at [`frontend/`](./frontend)

## Backend

```
./gradlew bootRun
```

## Frontend

```
cd frontend
npm install
npm run dev
```
